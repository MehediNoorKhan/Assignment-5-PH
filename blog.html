Question-1: What are the different ways to select an element in the DOM?

The ways to select an element in the DOM are:
getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content inside an element, innerText gets or sets the visible text and textContent gets or sets all text inside an element (including hidden text)

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single parent element handles events for its child elements, using event bubbling to capture events that originate from descendants.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts on the element clicked, and then it bubbles upward to all its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an element:

const newElement = document.createElement('p');
newElement.textContent = 'DevBoard!';

Adding an element:

document.body.appendChild(newElement);

Removing an element:

document.body.removeChild(newElement);
